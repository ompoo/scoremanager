# Technology Stack

このプロジェクトで使用されている主要な技術スタックです。

## Core
- **Framework**: Next.js 16.0.10 (App Router)
- **Library**: React 19.2.1
- **Language**: TypeScript

## Database & Backend
- **Platform**: Supabase
- **SDK**: `@supabase/supabase-js`, `@supabase/ssr`

## Styling
- **CSS Framework**: Tailwind CSS v4
- **Tools**: PostCSS

## Utilities
- **Validation**: zod
- **URL State**: nuqs

## Package Manager
- npm
