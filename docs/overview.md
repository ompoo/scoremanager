# Project Overview: Score Manager

## 概要
Score Managerは、楽譜（本）や楽曲を管理・検索するためのWebアプリケーションです。
Next.js (App Router) と Supabase を使用して構築されています。

## 主な目的
- ユーザーが所有する楽譜（Book）の管理
- 楽譜に含まれる楽曲（Song）の検索
- 詳細な検索条件によるフィルタリング

## 技術スタック
- **Frontend**: Next.js, React, Tailwind CSS
- **Backend/Database**: Supabase
- **Language**: TypeScript
